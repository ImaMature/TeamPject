## 보완할 점
> 콘솔창으로 하는 프로젝트이다보니, 한개의 콘솔창으로만 출력, 출력하는 데 한계가 있음을 발견(실제 PC방에서는 관리자는 관리자화면 따로, 손님은 손님화면 따로 . 나중에 DB연동해서 UI까지 사용하면 더 제대로 된 프로그램이 나올듯.
